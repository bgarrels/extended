DESCRIPTION

TBiDiDBNavigator is a descendent of standard Delphi's TDBNavigator. In addition to TDBNavigator functionality and featurs, this component supports custom glyphs for the navigation buttons and bi-directional mode.


KEY PROPERTIES

  • BiDiMode: TBiDiMode
    Specifies the bi-directional mode of the control.

  • ParentBiDiMode: Boolean
     Specifies whether the control uses its parent's BiDiMode.

  • Transparent: Boolean
    Specifies whether controls that sit below the navigator on a form can be seen through the navigator.

  • GlyphXXXXX: TBitmap
    Determines the image assocciated to the related button of the navigator.
